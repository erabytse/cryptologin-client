# Contributing to CryptoLogin Client

Thank you for your interest in contributing!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/cryptologin-client.git`
3. Install dependencies: `npm install`
4. Run tests: `npm test`

## Pull Request Process

1. Create a feature branch: `git checkout -b feature/my-feature`
2. Write tests for your changes
3. Ensure all tests pass: `npm test`
4. Commit your changes: `git commit -m 'Add some feature'`
5. Push to the branch: `git push origin feature/my-feature`
6. Open a Pull Request

## Code Style

- Use ES6+ modules
- Write JSDoc comments for all public functions
- Keep functions small and focused
- Write tests for new features

## Reporting Bugs

Open an issue at [GitHub Issues](https://github.com/erabytse/cryptologin-client/issues) with:

- Clear description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Environment (Node.js version, browser, etc.)
